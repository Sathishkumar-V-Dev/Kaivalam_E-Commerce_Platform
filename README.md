KAIVALAM – Java E-Commerce Web Application.
KAIVALAM is a full-stack Java-based e-commerce web application for selling handcrafted artisan products such as clay pots, diyas, and eco-friendly Ganesha idols.
The application implements complete shopping flow including authentication, cart management, order placement, and order history tracking.

@ Features:
# User Registration & Login (Session-based authentication)
# Dynamic Navbar (Login/Logout, Cart, My Orders)
# Add to Cart with quantity management
# Session-based cart storage
# Checkout with shipping address & payment mode (UI)
# Transaction-based order placement (Commit/Rollback)
# Automatic stock update after order
# Unique formatted Order ID
# Order history & detailed order view
# Clean invoice-style order summary

@ Tech Stack
# Backend: Java, JSP, Servlets, JDBC
# Database: MySQL
# Frontend: HTML, CSS, JavaScript
# Server: Apache Tomcat

@ Key Concepts Implemented
# MVC architecture using Servlets & JSP
# DAO Pattern
# Session Management
# Database Transactions
# Foreign Key Relationships
# Dynamic UI rendering
